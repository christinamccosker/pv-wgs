# Mapping
Map reads to the harbor seal reference genome.  

Chromosome-level reference genome available from DNA Zoo: https://www.dnazoo.org/assemblies/Phoca_vitulina  

**Scripts:**  
`02_bwa_index.sh`: Index the reference genome.  
`03_bwa.sh`: Map sample reads to the reference genome.  
`04_bwa_stat.sh`: Obtain read mapping statistics for each bam file.
